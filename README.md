# React Homepage Design

This project was built with Vite + React.

## Getting Started

1.  Open your terminal in this directory (`homepage-react`).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the local server URL (usually http://localhost:5173).

## Features
- **Monochrome & Vibrant Theme**: "Sports" inspired Electric Orange and Red accents.
- **Dark/Light Mode**: Fully functional toggle (top right) persisting to localStorage.
- **Component Based**: Modular architecture with Header, Hero, Features, and Footer components.
